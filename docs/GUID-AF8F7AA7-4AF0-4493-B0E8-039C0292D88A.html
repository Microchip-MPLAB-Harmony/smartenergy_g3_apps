1.8.3.3 WBZ451 Curiosity

PL460-EK is connected to mikroBUS connector (J4) of WBZ451 Curiosity using a mikroBUS to Xplained Pro adapter (connections shown in the table below). A 15V DC jack must be plugged into PL460-EK in order to supply the power for PLC transmission. For more information about PL460-EK refer to its user guide.

Sleep Mode and Thermal Monitor functionalities are not used in this platform, as there are not enough GPIOs on mikroBUS connector to support them. Sleep Mode pin is set to GND in the adapter to avoid issues.

WBZ451 has an ADC peripheral which is not currently supported by Pvdd Monitor service, thus this funcitonality is not added to this platform projects.

WBZ451 PHY layer is limited to packets up to 128 bytes. To comply with this limitation, G3 projects for this platform limit fragmentation size at 6LowPAN level to 100 bytes.
The micro-B USB connector (J7) of WBZ451 Curiosity can be used to:
  • Show debug information and information about the cycles, using a terminal emulator (console). UART baud-rate is configured at 115200 bps.
  • Flash/debug the WBZ451 device with the integrated programmer and debugger MPLAB PICkit On-Board 4 (PKOB4).

For more information about WBZ451 Curiosity refer to its user guide.

Note: WBZ451 radio is not compatible with currently supported modes on projects for RF215, so nodes of those platforms cannot be mixed in the same network.
