1.7 G3 Device UDP/ICMPv6 Responder

The G3 Device UDP/ICMPv6 Responder application implements a G3-Hybrid Device, capable of joining a G3 network. Once it is joined it can reply to UDP and ICMPv6 requests.

This application can be used to pass the G3 Conformance certification tests:
  • G3 stack is configured according to the requirements of these tests.
  • The implemented UDP responder supports the commands required for these tests.
  • It replies to ICMPv6 Echo Requests, which is required for these tests.

The provided projects are PLC-RF hybrid, but they can be PLC-only or RF-only if only one expansion board is connected. There are projects with FreeRTOS and without operative system (Bare-Metal). The following table shows the available G3 Device UDP/ICMPv6 Responder projects:
